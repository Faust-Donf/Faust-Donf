Current focus:

- Claude Code / Codex / ChatGPT / MCP / Obsidian 相关的 AI 工作流工具
- 统计推断、机器学习、量化研究和数据可视化
- 云计算与轻量生产系统：部署清楚、边界明确、验证路径可复现
